Animate A Gradient

First add the h1 element to your HTML page and give it a class attribute. Target the class in CSS and give the following properties.

HTML:

<h1 class="gradient-title"> Title </h1>

CSS:

.gradient-title {

  font-size: 110px;

  background: linear-gradient(#6dc1be, #a0d12e, #2ed16f);

  background-position: 0% 0%;

  background-size: 100% 500%;

  -webkit-text-fill-color: transparent;

  -webkit-background-clip: text;

  animation: gradient-animation 5s ease-in-out infinite alternate;

}

Animate A Gradient

Finally add the keyframes animation and we're done!

CSS:

@keyframes gradient-animation {

  0% {

    background-position: 0% 0%;

  }

  100% {

    background-position: 0% 100%;

  }

}
